

Installation Instructions:
==================================================
1. Import the "SQL/codeigniter_db.sql" file into the database of your CodeIgniter application.
2. Move all files to the same directory of your CodeIgniter application.
3. Open the "application/config/stripe.php" file ===> Specify the Stripe API Secret key (stripe_api_key), API Publishable key (stripe_publishable_key), and currency code (stripe_currency).
4. Open the URL (https://www.example.com/subscription/) on the browser and test the Stripe Subscription payment functionality in CodeIgniter.
